# 3D Tank Battle (3D 坦克大战)

这是一个基于 WebGL 的 3D 坦克对战游戏，使用 **TypeScript** 和 **Three.js** 开发。玩家将在广阔的 3D 战场中驾驶坦克，利用障碍物掩护，拾取强力道具，与智能 AI 坦克进行激烈的战斗。

## 📖 背景故事

这个项目的灵感来源于我看到我家小孩在海信电视上玩的一款 3D 坦克游戏，画面和玩法都很有趣。这让我想起了自己小时候玩红白机（FC）经典游戏《坦克大战》的快乐时光。

为了重温这份回忆，也为了给孩子做一个可以在电脑网页上随时玩的版本，我开发了这个 3D 版本的坦克对战游戏。目前的版本是一个自由对战模式，未来我计划加入像经典《坦克大战》那样的关卡系统，让这份情怀延续下去。

## 🎮 游戏特色

- **3D 沉浸体验**：完全 3D 化的场景和坦克模型，支持第三人称跟随视角。
- **智能 NPC**：敌人拥有巡逻、追击和攻击等多种行为模式，会根据战况做出反应。
- **丰富的道具系统**：
  - 🛡️ **防护罩 (Cyan)**：获得 10 秒无敌护盾。
  - 🚀 **火箭弹 (Red)**：15 秒内攻击力大幅提升。
  - 💚 **急救包 (Green)**：恢复坦克生命值。
- **动态环境**：广阔的地图（200x200），随机生成的障碍物掩体，以及迷雾效果。
- **视听特效**：包含粒子爆炸特效和基于 Web Audio API 合成的动态音效。

## 🛠️ 技术栈

- **核心引擎**: [Three.js](https://threejs.org/) - 强大的 3D 图形渲染库。
- **开发语言**: [TypeScript](https://www.typescriptlang.org/) - 提供类型安全的开发体验。
- **构建工具**: [Vite](https://vitejs.dev/) - 极速的前端构建工具。
- **物理/碰撞**: 自研的轻量级 AABB 碰撞检测系统。

## 🚀 快速开始

### 环境要求

- Node.js (推荐 v16+)
- npm 或 yarn

### 安装步骤

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/3d-tank-battle.git
   cd 3d-tank-battle
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

4. 打开浏览器访问 `http://localhost:5173` 即可开始游戏。

## 🕹️ 操作说明

- **移动**: `W`, `A`, `S`, `D` 或 `↑`, `↓`, `←`, `→`
- **瞄准**: 移动鼠标控制炮塔旋转
- **射击**: 点击鼠标左键
- **暂停/继续**: `ESC` 键
